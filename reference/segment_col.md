# Accessor Functions to Obtain Segment / Group Column

The name of the column in the fit where the newly created segmentation
or grouping is stored.

## Usage

``` r
segment_col(x, ...)

group_col(x, ...)
```

## Arguments

- x:

  the object from which the column name should be extracted.

- ...:

  additional arguments currently ignored.

## Value

a character vector giving the name of the column where the output is
stored.
